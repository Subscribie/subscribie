Synchronise Data from Stripe

What does this do?

This tool synchronises data from Stripe to your shop. It's useful for:

  • Making sure your shop has the latest information from Stripe
  • Recovering from any missed updates
  • Ensuring your records match Stripe

Note: This is safe to run and won't create duplicates if run multiple times.

Enter a number between 1 and 365 days. Default is 30 days.
All types are selected by default. Uncheck any you don't want to backfill.
How It Works

This process retrieves data from Stripe for the specified time period and ensures your shop's records are up to date and match what's in Stripe.

Tips

Runs in background: The synchronisation happens behind the scenes, so you can continue using your shop while it completes.

Recommended: Backfill all data types together to ensure consistency.

Safe to re-run: You can run this multiple times without creating duplicates.
